The main Matrix library header file. A simple Matrix operation library, to Multiply 2 Matrices and find the transpose of a Matrix. The library is divided into two parts, one for small Matrices (i.\+e. Matrix multiplication is performed with O($^\wedge$3) time complexity) and another for large Matrices where a mixture of Stressan\textquotesingle{}s Algorithm (Time Complexity\+: O(n$^\wedge$2.8)) and the naive algorithm (Time Complexity\+: O(n$^\wedge$3)) is used to Multiply two Matrices.
