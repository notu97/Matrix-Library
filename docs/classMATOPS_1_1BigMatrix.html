Detailed Description

template<typename Data1>
class MATOPS::BigMatrix< Data1 >

This is the Class for handling Large Matrices. It takes in large Matrices as comma-separated values (CSV) files and perform both Multiplication (Stressan's Algorithm) and transpose.

Template Parameters
Data1= Datatype of the BigMatrix. Eg. int, float, double etc.

This is the BigMatrix multiplication Function. To multiply two matrices A and B stored in A.csv and B.csv respectively and store the result in C.csv file, do the following.

Parameters
file_1= "path to A.csv"
file_2= "path to B.csv"
path= path to a store file (no need to predefine C.csv file in the directory, it gets generated automatically.)
print= True, To see all the Matrices i.e. A,B and C in the output terminal/stdio.
Returns
Returns a pointer of type Data1** pointing to the C BigMatrix in the Memory. It can be used in the main program to access the (i,j) element of the BigMatrix C.

Define a BigMatrix object BigMatrix<DataType> MatObj, where DataType can be int, float, double, long etc.
Then call DataType** Result = MatObj.matmul("path/to/A.csv","path/to/B.csv","path/to/C.csv") from your program. The C.csv file generated file contains C=A*B. Result[i][j] accesses the (i,j)th element of BigMatrix C.
